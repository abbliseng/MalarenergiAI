# MalarenergiAI
Litet AI projekt där man ska förutspå värden på saker :)
## Planen
En AI modell som kan förutspå förbrukningen i ett husområde utifrån temperatur och årstid.
### Delmål
1.  Läsa in data
3.  Formatera data
4.  Lista ut AI delen
5.  Utvärderade sannolikheten
6.  Presentera svar, grafiskt.
